#numgess

Get user's name and Guess random number 

## Installation
```shell
$ pip install
```

## How to start

if you want to 'emphasize' some text, just use backquotes.

```shell
$ python numguess.py
```

## Features

- Get user name
- Set answer
- Get user's guess
- Compare answer with user's guess
- Give little surprise.
